I set up a space for myself in the basement. This is now where I keep my desktop — running Ubuntu at the moment. Easy enough to get at from the laptop upstairs. Maybe keep Irssi running in a Tmux session or something.

Oh, and just in case you want to know how the basement looked when we moved in, here’s a picture.

I like it better now.

What have I been coding in this workspace that looks a bit less like a horror movie scene than it did a few weeks ago?

I’m working on a test rule for this site. Mainly I use it to check for bad links in an attempt to deal with link rot. The link checker is Python with Requests, and seeing if Scrapy can be useful in this context. Scrapy seems like overkill at first glance, but it has middleware for respecting robots.txt. I care about being a good Internet citizen.

I plan to write about that soon, since I enjoy the task so far.
