Obsidian Markdown

January 21, 2024

Special considerations and features that I care about for Markdown in Obsidian

Linking

  • Internal links with [[link]]
  • Internal images with regular ![](link) or ![[link]]
  • YouTube embeds with ![](link)

Callouts

aka admonitions

Officially supported types. Each entry gets its own rendering style. Any unrecognized type is treated as Note.

  • Note
  • Abstract / summary / tldr
  • Info
  • Todo
  • Tip / Hint / Important
  • Success / Check / Done
  • Question / Help / Faq
  • Warning / Caution / Attention
  • Failure / Fail / Missing
  • Danger / Error
  • Bug
  • Example
  • Quote / Cite
